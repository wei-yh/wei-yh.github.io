我的博客

此信息用来测试将hexo环境备份到GitHub - wei-yh/blog上并上传。后续会用来记录下此次博客如何搭建成功，并且部署和备份的。以及在一台新的环境的电脑上如何发布BLOG的。

前置配置：

  • install Node.js https://nodejs.org/en/: 为了安装包管理器npm
  • install -g hexo-cli
# npm install hexo #我安装失败了由于众所周知的网络问题
# 使用淘宝镜像
npm install -g cnpm --registry=https://registry.npm.taobao.org
cnpm install -g hexo-cli
  1. clone 备份项目 blog
    git clone https://github.com/wei-yh/blog.git
  1. 修改过往或者新建blog
  2. hexo 命令重新generate 和 deployed
    hexo clean
    hexo g
    hexo d
